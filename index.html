Pregunta 1: Escriu la sintaxi bàsica d'un bloc try...catch.

La declaració try…catch assenyala un bloc d’instruccions a intentar (try), i específica una resposta si es produeix una excepció (catch).

try { nonExistentFunction(); } catch (error) { console.error(error); }

Pregunta 2: Per comprovar si una petició fetch ha anat bé abans de llegir les dades, cal mirar la propietat response.ok o el response.status.

La declaració try…catch assenyala un bloc d’instruccions a intentar (try), i específica una resposta si es produeix una excepció (catch).

Response.ok (millor pràctica): fetch("products.json") .then((response) => { if (!response.ok) { throw new Error(`HTTP error: ${response.status}`); } return response.json(); }) .then((json) => initialize(json)) .catch((err) => console.error(`Fetch problem: ${err.message}`)); Response.status: fetch('https://api.exemple.com/dades') .then(response => { if (response.status !== 200) { throw new Error(`Codi incorrecte: ${response.status}`); } return response.json(); }) .then(data => console.log(data)) .catch(error => console.error(error));
